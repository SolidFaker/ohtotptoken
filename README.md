## Feature

- [x] Support TOTP Token
- [x] Support HOTP Token
- [x] Support Steam Token, [Importing from Steam](https://github.com/stratumauth/app/wiki/Importing-from-Steam)
- [x] Support Forti Token
- [x] Setting Page
- [ ] Asset Store Kit
